Step 1: Set up folder sharing on your PC

We need to expose a local folder so the Android app can upload firmware files into it.

⚡ Ubuntu / Linux Quick Share Setup

This will automatically configure Samba on your Linux machine and register it to the coordination server.

📁 Windows folder sharing guidelines

On Windows, you can share a folder natively via File Explorer:

  1. Right click your target folder → Properties.
  2. Go to the Sharing tab → click Share....
  3. Select your user account, grant Read/Write access, and click Share.
  4. Note down your PC's LAN IP address and shared folder credentials to enter in Step 2.

🌐 Synchronization Coordination Server

Enter the credentials of your coordination hub to pair your devices automatically.

Step 2: Sync and Link your Android device

Pair your Android phone with the PC configurations using the cloud bridge.

📲 How to Sync:

  1. Open the FireFiles Bridge app on your Android device.
  2. Ensure you have set the matching Monitor URL and Token.
  3. Press the Fetch Config from Server button in the Android App.
  4. All Samba configuration fields on the phone will auto-fill themselves!

Step 3: Specify local folders and start syncing

Run the synchronization daemon to copy files from the shared inbox to your backup directory.
